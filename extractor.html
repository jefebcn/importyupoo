▼ Come ottenere il Worker URL (gratis, 2 minuti)
  1. Vai su dash.cloudflare.com → Workers & Pages → Create
  2. Scegli "Hello World", dai un nome (es. yupoo-proxy), clicca Deploy
  3. Clicca Edit code, cancella tutto, incolla il codice dal file yupoo-proxy-worker.js
  4. Clicca Deploy, copia l'URL (es. https://yupoo-proxy.pippo.workers.dev)
  5. Incolla l'URL nel campo qui sopra — il download ZIP funzionerà
